Setup Docker service to convert media (mp4) to RTSP -
docker run --rm -it -v ./others/mediamtx.yml:/mediamtx.yml -p 18554:18554 bluenviron/mediamtx:latest-ffmpeg

Setup Ffmpeg to post media (mp4) to RTSP hosted by Docker -
ffmpeg -re -stream_loop -1 -i ./others/straight-line-walking.mkv -c copy -f rtsp -rtsp_transport tcp rtsp://Localhost:18554/mystream -loglevel verbose
